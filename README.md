# 🛍️ Bubu Collection

فروشگاه آنلاین لباس - ساخته شده با Node.js و SQLite

## 🛠️ تکنولوژی‌ها
- **Backend:** Node.js, Express
- **Database:** SQLite, Sequelize
- **Frontend:** HTML, CSS, JavaScript

## ✨ قابلیت‌ها
- نمایش محصولات
- پنل ادمین با احراز هویت
- افزودن، ویرایش و حذف محصول
- ریسپانسیو

## 🚀 نصب و اجرا
```bash
npm install
node server.js
```
